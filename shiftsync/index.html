problem

こんなお悩みありませんか？

✍️
手書きの○付けが大変

「常勤・非常勤」「専従・兼務」を一人ずつ手書きで○…。何十人分もやると腱鞘炎になりそう。

⏰
転記に数時間かかる

KingOfTimeの画面を見ながらExcelに転記。1ヶ月分×全職員で、毎月数時間が消えていく。

😰
ミスが怖い

シフト記号の入力ミス、常勤と非常勤の判定ミス…。行政提出書類だからこそ間違えられない。

📊
複数フォーマット対応

通所介護、訪問介護、一般…。フォーマットが違う書類を毎月何種類も作成しなければならない。

solution

ShiftSyncなら、こう変わります

Before（従来）
  • 1️⃣KingOfTimeにログイン
  • 2️⃣シフトを目視で確認
  • 3️⃣Excelに手入力
  • 4️⃣常勤/非常勤に○付け
  • 5️⃣集計を手計算

  • ✅そのまま提出！

features

ShiftSyncの4つの特長

🔄
シフト記号
自動変換

KingOfTimeの出退勤データをＢ1・Ｂ2・P半・公休などのシフト記号に自動変換します。

⭕
常勤/非常勤
自動○付け

KOTの雇用区分から常勤・非常勤、専従・兼務を自動判定。手書きの○付けとサヨナラ。

📄
テンプレート
そのまま

お使いのExcelフォーマットをそのまま使用。数式も書式も結合セルもすべて保持します。

🖱️
かんたん
操作

ファイルをダブルクリック → ブラウザが開く → ボタンを押す。それだけで完了です。
