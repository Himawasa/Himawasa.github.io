1❌ 15分放置で30秒スリープ✅ Cloud Run → スリープなし（0.5〜2秒で応答）

5❌ ビジネスメールなし✅ Google Workspace → 独自ドメインのメール

After: ポータルの「取込」ボタンを押すだけ → 自動取得（数秒）
