まず「クラウド」って何？

クラウドとは、インターネット上にある「レンタルサーバー」のことです。

自分でサーバー（コンピューター）を買わなくても、月額料金を払うだけでアプリやデータベースを動かせます。

💡 たとえるなら… 自社ビルを買う → ❌ 高い・管理が大変
レンタルオフィスを借りる → ✅ 安い・掃除や修繕はビル管理会社がやってくれる

クラウド = ITのレンタルオフィスです。

世界には3つの大きなクラウド会社があります。

クラウド名運営会社ひとことで言うと
GCP
（Google Cloud）		GoogleGmail・YouTube を動かしている技術
AWS
（Amazon Web Services）		AmazonAmazonの通販サイトを動かしている技術
Azure
（アジュール）		MicrosoftWordやExcelのクラウド版を動かしている技術

比較する4つのポイント

社労士ポータルを動かすには、以下の4つが必要です。

#必要なものたとえるなら
1🖥️ アプリを動かす場所レンタルオフィスの「部屋」
2🗄️ データベース
（データの保管庫）		書類を保管する「金庫」
3🔒 固定の住所（固定IP）
（KOT API連携に必須）		郵便物が届く「住所」
4⚖️ 受付（ロードバランサー）
（アクセスを振り分け）		ビルの「受付嬢」
⚠️ なぜ「固定の住所」が必要？ KOT（勤怠管理システム）は、セキュリティのため「この住所からのアクセスだけ許可」という仕組みです。
住所がコロコロ変わると、KOTに「あなた誰？」と拒否されます。
だから固定の住所（固定IP）が必須なのです。

2. 使わない時間の課金はどうなる？

社労士ポータルは1日のアクセスが100回程度。深夜や休日は誰も使いません。
使わない間も料金がかかるかどうかは、とても大事なポイントです。

項目GoogleAmazonMicrosoft
誰も使ってない時¥0（自動停止）課金され続ける¥0（自動停止）
再開までの待ち時間0.5〜2秒なし（常に動いてるので）1〜3秒
💡 たとえるなら… Google・Microsoft = 必要な時だけ電気をつける → 省エネ
Amazon = 24時間電気つけっぱなし → もったいない

小規模なポータルでは、使った分だけ払える方がお得です。

4. 3社の得意・不得意

🟢 Google Cloud（GCP）

👍 得意なこと👎 苦手なこと
✅ AIが一番強い（Gemini・生成AI）❌ Amazonほど「みんなが使ってる」感はない
✅ Gmail・Googleドライブと同じアカウントで管理❌ 機能の種類はAmazonより少ない
✅ 小規模なら圧倒的に安い
✅ 画面がわかりやすい（操作が直感的）
✅ 固定IPが安い

🟠 Amazon（AWS）

👍 得意なこと👎 苦手なこと
✅ 世界で一番使われている（業界No.1）❌ 固定IPが異常に高い（月¥7,346）
✅ できることが一番多い（200以上のサービス）❌ 料金の仕組みが複雑でわかりにくい
✅ ネットに情報がたくさんある❌ 使わなくても課金される（常時稼働前提）
❌ Google Workspaceとの連携はできない

🔵 Microsoft（Azure）

👍 得意なこと👎 苦手なこと
✅ WordやExcel（Microsoft 365）と相性抜群❌ 固定IPがやや高い（月¥5,413）
✅ ChatGPT（OpenAI）を法人利用できる❌ Google Workspaceとの連携はできない
✅ 使わない時は自動停止（省エネ）❌ 画面が少し複雑

「Microsoftでもいいのでは？」への回答

💡 Microsoftが向いているのは… 会社でWord・Excel・Teamsを中心に使っていて、その延長でクラウドも統一したい場合。
今回はGoogle Workspace（Gmail・Googleドライブ）を使うので、Googleで統一する方が自然です。

「Amazonが業界No.1なのに選ばないの？」への回答

⚠️ Amazonが向いているのは… 大企業が大量のサーバーを管理する場合。Netflix や Airbnb のような大規模サービス向き。
今回のような小規模ポータルでは「使ってない時も課金」「固定IPが高い」でコスパが悪い。
